REST API & Integrations

Integrate AquaVision's underwater enhancement pipeline into your own applications. Create an access token in the API Keys Dashboard.

POST /api/v1/enhance

Enhance a single underwater image. Returns classification, quality metrics, improvement percentage, and the enhanced image as a base64-encoded JPEG.

Headers & Parameters
NameTypeDescription
Authorization headerStringBearer <your_api_key>
file requiredFileImage file (JPEG, PNG, BMP, TIFF)
strength optionalStringmild | standard (default) | strong
format optionalStringjpeg (default) | png | tiff
cURL
curl -X POST http://localhost:5000/api/v1/enhance \ -H "Authorization: Bearer YOUR_API_KEY" \ -F "file=@underwater_photo.jpg" \ -F "strength=standard" \ -F "format=jpeg"
Python
import requests, base64 from PIL import Image from io import BytesIO headers = {"Authorization": "Bearer YOUR_API_KEY"} resp = requests.post( "http://localhost:5000/api/v1/enhance", headers=headers, files={"file": open("underwater.jpg", "rb")}, data={"strength": "standard", "format": "jpeg"} ) data = resp.json() print(f"Degradation: {data['prediction']}") print(f"Confidence: {data['confidence']}%") print(f"Improvement: +{data.get('improvement_pct', 'N/A')}%") img_bytes = base64.b64decode(data["enhanced_image_base64"]) Image.open(BytesIO(img_bytes)).save("enhanced_output.jpg")
